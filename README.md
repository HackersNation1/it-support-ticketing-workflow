# it-support-ticketing-workflow# IT Support Ticketing Workflow (Beginner Lab)

This repository documents a simple IT support and help desk workflow
built using Google Forms and Google Sheets.

The goal of this project is to practice real-world IT support processes
such as ticket intake, diagnosis, status tracking, and resolution â€”
similar to entry-level help desk environments.

---

## Tools Used
- Google Forms (ticket intake)
- Google Sheets (ticket tracking and status management)
- Windows OS
- Basic networking tools (ipconfig, ping, DNS checks)
- Malwarebytes (malware detection)

---

## Ticket Workflow
1. User submits issue via Google Form
2. Ticket is logged automatically in Google Sheets
3. Ticket is reviewed and diagnosed
4. Status is updated:
   - New
   - In Progress
   - Waiting for Payment
   - Resolved
   - Closed
5. Resolution details are recorded for reference

---

## Issues Handled
- Software installation and configuration
- Malware and virus removal
- Wi-Fi and network connectivity issues
- System cleanup and optimization

> Note: This project covers software-level support only.
> No hardware repairs are performed.
