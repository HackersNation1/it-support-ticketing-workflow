# Common IT Support Issues and Diagnosis

This document lists common software-level issues encountered
during basic IT support and how they are diagnosed.

---

## Slow System Performance
**Possible Causes**
- Too many startup programs
- Malware
- Low disk space

**Diagnosis Steps**
- Check Task Manager for high resource usage
- Scan system using Malwarebytes
- Review startup programs

---

## Malware / Virus Infection
**Possible Causes**
- Unsafe downloads
- Infected USB drives

**Diagnosis Steps**
- Run full system scan using Malwarebytes
- Quarantine detected threats
- Restart and rescan

---

## No Internet / Wi-Fi Issues
**Possible Causes**
- Incorrect IP configuration
- DNS issues
- Router problems

**Diagnosis Steps**
- Run `ipconfig`
- Ping default gateway
- Ping 8.8.8.8
- Ping domain names (e.g. google.com)

---

## Software Installation Issues
**Possible Causes**
- Corrupted installer
- Incompatible OS version
- License restrictions

**Diagnosis Steps**
- Verify system requirements
- Re-download installer
- Recommend free alternatives if needed
